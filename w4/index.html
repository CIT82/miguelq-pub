HTML Element 2

iframe

The iframe HTML element represents a nested browsing context, embedding another HTML page into the current one. In this case I'm embedding a youtube video from my youtube channel, that I took when I went to the Eras Tour at Sofi Stadium last year. The video was shot in 4K. The reason I didn't use the video html element tag is that for youtube videos, the video tag wont work for youtube videos. The video HTML element is meant for urls with file extensions like .mp4
